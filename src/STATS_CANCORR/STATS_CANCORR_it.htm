Calcolo delle correlazioni canoniche

Questa procedura calcola le correlazioni canoniche e le statistiche correlate per due insiemi di variabili. Facoltativamente calcola il punteggio dei dati e produce un file di sintassi che può essere utilizzato con i nuovi dati.

Insieme 1 Selezionare il primo insieme di variabili per le correlazioni.

Insieme 2 Selezionare il secondo insieme di variabili per le correlazioni.

Opzioni

Questa finestra di dialogo specifica quale output viene visualizzato e il comportamento del calcolo del punteggio.

Nome radice per variabili di punteggio Immettere un prefisso per i nomi delle variabili di punteggio generate.

• Sia che si selezioni il calcolo del punteggio sia che si selezioni la creazione di un file di sintassi per il calcolo del punteggio, è necessario indicare un nome radice.

Punteggi Selezionare la casella e specificare un nome dataset per poter creare un dataset di punteggi. Il dataset conterrà le variabili centrate e i punteggi.

File sintassi per il calcolo del punteggio Selezionare la casella e selezionare una specifica del file di output per creare un file di sintassi contenente i comandi per il calcolo del punteggio di un dataset. La sintassi centra i dati e calcola quindi i punteggi per entrambi gli insiemi di variabili.

• La sintassi prevede nomi di variabili uguali a quelli utilizzati per il calcolo delle correlazioni.

Limite dimensione per calcolo del punteggio Per impostazione predefinita tutte le correlazioni vengono utilizzate per il calcolo del punteggio. È possibile specificare un limite inferiore selezionando la casella e immettendo il numero di correlazioni canoniche da utilizzare.

Visualizza Selezionare le caselle per l'output desiderato. Le correlazioni pairwise sono uguali a quelle che sarebbero prodotte eseguendo la procedura Correlazioni con l'eliminazione listwise del valore mancante. Viene sempre prodotta una tabella di correlazioni e test.

Funzioni aggiuntive

Questa finestra di dialogo genera la sintassi del comando di estensione STATS CANCORR. Per visualizzare la guida per questo comando, utilizzare la sintassi riportata di seguito:

STATS CANCORR /HELP.

Requisiti

Questo comando richiede Python Essentials. È possibile scaricarlo tramite il sito Web SPSS Community all'indirizzo www.ibm.com/developerworks/spssdevcentral o reperirlo dal materiale di installazione.
