Вычислить канонические корреляции

Эта процедура вычисляет канонические корреляции и связанные статистические данные для двух наборов переменных. Дополнительно можно произвести скоринг данных и создать файл синтаксиса для использования с новыми данными.

Набор 1 Выберите первый набор переменных для корреляций.

Набор 2 Выберите второй набор переменных для корреляций.

Параметры

В этом диалоговом окне задается поведение скоринга и указывается, какие выходные данные будут показаны.

Корневое имя для скоринга переменных Введите префикс для имен сгенерированных переменных скоринга.

• Корневое имя требуется при выборе скоринга или при создании файла синтаксиса для скоринга.

Оценки Включите этот переключатель и укажите имя набора данных, чтобы создать набор данных оценок. Этот набор данных будет содержать центрированные переменные и оценки.

Файл синтаксиса скоринга Включите этот переключатель и выберите спецификацию выходного файла, чтобы создать файл синтаксиса, содержащий команды для скоринга набора данных. Этот синтаксис центрирует данные и затем вычисляет оценки для обоих наборов переменных.

• Для этого синтаксиса предполагается, что те же имена переменных использовались при вычислении корреляций.

Предел измерений скоринга По умолчанию для скоринга используются все корреляции. Включив этот переключатель и введя число используемых канонических корреляций, можно задать меньшее предельное значение числа корреляций.

Вывод на экран Включите эти переключатели для нужного вывода. Парные корреляции те же, что вычислялись бы процедурой Корреляции с удалением всех отсутствующих значений. Всегда создается таблица корреляций и критериев.

Дополнительные возможности

Это диалоговое окно формирует синтаксис для команды расширения STATS CANCORR. Чтобы показать справку синтаксиса, выполните следующий синтаксис:

STATS CANCORR /HELP.

Технические требования
