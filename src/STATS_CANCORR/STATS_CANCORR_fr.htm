Calcul de corrélations canoniques

Cette procédure calcule les corrélations canoniques et les statistiques associées pour deux ensembles de variables. Elle peut éventuellement calculer le score des données et générer un fichier de syntaxe utilisable avec de nouvelles données.

Ensemble 1 : Sélectionnez le premier ensemble de variables pour les corrélations.

Ensemble 2 : Sélectionnez le deuxième ensemble de variables pour les corrélations.

Options

Cette boîte de dialogue détermine les éléments de sortie ainsi que le comportement de calcul du score.

Nom de racine pour les variables de score : Attribuez un préfixe au nom des variables de score générées.

• Si l'option de calcul du score ou de création d'un fichier de score est sélectionnée, le nom de racine est obligatoire.

Scores : Cochez cette case et indiquez un nom de jeu de données afin de créer un jeu de données des scores. Le jeu de données contiendra les variables centrées et les scores.

Fichier de syntaxe pour le calcul du score : Cochez cette case et sélectionnez une spécification de fichier de sortie afin de créer un fichier de syntaxe qui contient les commandes de calcul du score d'un jeu de données. La syntaxe centre les données, puis calcule les scores pour les deux ensembles de variables.

• La syntaxe attend les mêmes noms de variables que ceux utilisés pour le calcul des corrélations.

Limite de dimensions pour le calcul du score : Par défaut, toutes les corrélations sont utilisées pour le calcul du score. Vous pouvez définir une limite inférieure en cochant cette case et en entrant le nombre de corrélations canoniques à utiliser.

Affichage : Cochez les cases correspondantes aux éléments de sortie souhaités. Les corrélations par paires sont les mêmes que celles obtenues par la procédure des corrélations avec suppression de la valeur manquante de l'observation incomplète. Un tableau des corrélations et des tests est toujours généré.

Fonctions supplémentaires

Cette boîte de dialogue génère la syntaxe de la commande d'extension STATS CANCORR. Pour afficher l'aide sur la syntaxe de cette commande, exécutez la syntaxe suivante :

STATS CANCORR /HELP.

Conditions requises

Cette commande requiert Python Essentials. Vous pouvez le télécharger depuis le site Web de la communauté SPSS à l'adresse www.ibm.com/developerworks/spssdevcentral ou à partir de vos supports d'installation.
