正準相関の計算

この手順では、2 つの変数グループの正準相関および関連する統計値を計算します。オプションとして、データをスコアリングできるほか、新規のデータで使用できるシンタックス ファイルを作成できます。

グループ 1: 相関の一方の変数グループを選択します。

グループ 2: 相関のもう一方の変数グループを選択します。

オプション

このダイアログでは、表示する出力やスコアリング動作を指定します。

スコア変数のルート名: 生成されるスコア変数の名前に付加する接頭辞を入力します。

• スコアリングを行う場合、またはスコアリング用のシンタックス ファイルの作成を行う場合は、ルート名が必要です。

スコア: スコアのデータセットを作成するには、このボックスにチェック マークを付け、データセット名を指定します。 データセットには、中心化された変数とスコアが含まれることになります。

スコアリング シンタックス ファイル: データセットをスコアリングするためのコマンドを含んだシンタックス ファイルを作成するには、このボックスにチェック マークを付けて出力ファイルの仕様を選択します。シンタックスでは、データを中心化し、両方の変数グループのスコアを計算します。

• シンタックスでは、相関の計算に使用されたものと同じ変数名を使用する必要があります。

スコアリング次元の制限: デフォルトでは、すべての相関がスコアリングに使用されます。このボックスにチェック マークを付け、使用する正準相関の数を入力することで、制限を高めることができます。

表示: 必要な出力に対してこのボックスにチェック マークを付けます。 ペアごとの相関係数は、リストごとの欠損値削除で相関手順を実行して生成されるものと同じです。相関と検定のテーブルが常に生成されます。

その他の機能

このダイアログは、STATS CANCORR 拡張コマンドのシンタックスを生成します。このコマンドのシンタックスのヘルプを表示するには、次のシンタックスを実行します。

STATS CANCORR /HELP.

要件

このコマンドには Python Essentials が必要です。 Python Essentials は SPSS Community Web サイト (www.ibm.com/developerworks/spssdevcentral) からダウンロードできるほか、インストール資料にも収録されています。
