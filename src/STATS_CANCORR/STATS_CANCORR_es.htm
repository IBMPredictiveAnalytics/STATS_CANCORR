Calcular correlaciones canónicas

Esta procedimiento calcula las correlaciones canónicas y los estadísticos relacionados de dos conjuntos de variables. Opcionalmente, puede puntuar los datos y producir un archivo de sintaxis que se puede utilizar con datos nuevos.

Conjunto 1 Seleccione el primer conjunto de variables para las correlaciones.

Conjunto 2 Seleccione el segundo conjunto de variables para las correlaciones.

Opciones

Este diálogo especifica el resultado que se visualizará y el comportamiento de puntuación.

Nombre raíz para las variables de puntuación Especifique un prefijo para los nombres de las variables de puntuación generadas.

• Si se selecciona puntuación o crear un archivo de sintaxis para puntuación, se requiere un nombre raíz.

Puntuaciones Marque el recuadro y especifique un nombre de conjunto de datos para crear un conjunto de datos de puntuaciones. El conjunto de datos contendrá las variables centradas y las puntuaciones.

Archivo de sintaxis para puntuación Marque este recuadro y seleccione una especificación de archivo de salida para crear un archivo de sintaxis que contenga los comandos para puntuar un conjunto de datos. La sintaxis centra los datos y después calcula las puntuaciones para los dos conjuntos de variables.

• La sintaxis espera los mismos nombres de variables que se han utilizado para calcular las correlaciones.

Límite de dimensión de puntuación De forma predeterminada, todas las correlaciones se utilizan para la puntuación. Puede especificar un límite menor marcando el recuadro y especificando el número de correlaciones canónicas a utilizar.

Mostrar Marque los recuadros según el resultado que desee obtener. Las correlaciones por parejas son las mismas que se producirían ejecutando el procedimiento Correlaciones con supresión de valores perdidos por orden de lista. Siempre se genera una tabla de correlaciones y pruebas.

Características adicionales

Este diálogo genera sintaxis para el comando de extensión STATS CANCORR. Para mostrar la ayuda de sintaxis de este comando, ejecute la siguiente sintaxis:

STATS CANCORR /HELP.

Requisitos

Este comando requiere Python Essentials. Puede descargarlos del sitio web de la Comunidad de SPSS, en www.ibm.com/developerworks/spssdevcentral o de los materiales de instalación.
