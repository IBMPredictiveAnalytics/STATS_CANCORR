Calcular correlações canônicas

Este procedimento calcula correlações canônico e estatísticas relacionadas para dois conjuntos de variáveis. Opcionalmente, ele pode escorar os dados e produzir um arquivo de sintaxe que pode ser usado com novos dados.

Conjunto 1 Selecione o primeiro conjunto de variáveis para as correlações.

Conjunto 2 Selecione o segundo conjunto de variáveis para as correlações.

Opções

Este diálogo especifica qual saída é exibida e o comportamento de escoragem.

Nome raiz para variáveis de escore Insira um prefixo para os nomes das variáveis de escore geradas.

• Se escoragem ou criar um arquivo de sintaxe para escoragem for selecionado, um nome raiz é necessário.

Escores Marque a caixa e especifique um nome de conjunto de dados para criar um conjunto de dados de escores. O conjunto de dados irá conter as variáveis centralizadas e os escores.

Arquivo de sintaxe de escoragem Marque a caixa e selecione uma especificação de arquivo de saída para criar um arquivo de sintaxe que contém os comandos para escoragem de um conjunto de dados. A sintaxe centraliza os dados e, em seguida, calcula escores para ambos os conjuntos de variáveis.

• A sintaxe espera os mesmos nomes de variáveis que foram usados para calcular as correlações.

Limite de dimensão de escoragem Por padrão, todas as correlações são usadas para escoragem. É possível especificar um limite menor marcando a caixa e inserindo o número de correlações canônicas para usar.

Exibir Marque as caixas para a saída desejada. As correlações de pares são as mesmas que seriam produzidas pela execução do procedimento Correlações com exclusão de valor omisso de lista. Uma tabela de correlações e testes sempre é produzida.

Recursos adicionais

Este diálogo gera sintaxe para o comando de extensão STATS CANCORR. Para exibir ajuda de sintaxe para este comando, execute a seguinte sintaxe:

STATS CANCORR /HELP.

Requisitos

Este comando requer o Python Essentials. É possível fazer download dele por meio do website SPSS Community em www.ibm.com/developerworks/spssdevcentral ou de seus materiais de instalação.
