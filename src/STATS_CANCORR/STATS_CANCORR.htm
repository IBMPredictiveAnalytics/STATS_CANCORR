Compute Canonical Correlations

This procedure computes canonical correlations and related statistics for two sets of variables. Optionally it can score the data, and it can produce a syntax file that can be used with new data.

Set 1 Select the first set of variables for the correlations.

Set 2 Select the second set of variables for the correlations.

Options

This dialog specifies what output is displayed and the scoring behavior.

Rootname for Score Variables Enter a prefix for the names of the generated score variables.

• If either scoring or creating a syntax file for scoring is selected, a rootname is required.

Scores Check the box and specify a dataset name in order to create a dataset of scores. The dataset will contain the centered variables and the scores.

Scoring Syntax File Check the box and select an output file specification to create a syntax file containing the commands for scoring a dataset. The syntax centers the data and then calculates scores for both sets of variables.

• The syntax expects the same variable names as were used for computing the correlations.

Scoring Dimension Limit By default all the correlations are used for scoring. You can specify a smaller limit by checking the box and entering the number of canonical correlations to use.

Display Check the boxes for the desired output. The pairwise correlations are the same as would be produced by running the Correlations procedure with listwise missing value deletion. A table of correlations and tests is always produced.

Additional Features

This dialog generates syntax for the STATS CANCORR extension command. To display syntax help for this command, run the following syntax:

STATS CANCORR /HELP.

Requirements

This command requires the Python Essentials You can download these via the SPSS Community website at www.ibm.com/developerworks/spssdevcentral or from your installation materials.
