Kanonische Korrelationen berechnen

Diese Prozedur berechnet kanonische Korrelationen und zugehörige Statistiken für zwei Variablensets. Optional kann mit ihr ein Scoring der Daten durchgeführt und eine Syntaxdatei erstellt werden, die bei neuen Daten verwendet werden kann.

Set 1 Wählen Sie das erste Variablenset für die Korrelationen aus.

Set 2 Wählen Sie das zweite Variablenset für die Korrelationen aus.

Optionen

In diesem Dialogfeld wird die Ausgabe, die angezeigt werden soll, und das Scoring-Verhalten angegeben.

Stammname für Scorevariablen Geben Sie ein Präfix für die Namen der generierten Scorevariablen ein.

• Wenn die Durchführung eines Scorings oder die Erstellung einer Syntaxdatei für ein Scoring ausgewählt wurde, ist ein Stammname erforderlich.

Scores Wählen Sie das Feld aus und geben Sie einen Datasetnamen an, um ein Dataset mit Scores zu erstellen. Das Dataset wird die zentrierten Variablen und die Scores enthalten.

Scoring-Syntaxdatei Wählen Sie das Feld und eine Ausgabedateispezifikation aus, um eine Syntaxdatei zu erstellen, die die Befehle für das Scoring eines Datasets enthält. Die Syntax zentriert die Daten und berechnet anschließend die Scores für beide Variablensets.

• Die Syntax erwartet dieselben Variablennamen, die auch bei der Berechnung der Korrelationen verwendet wurden.

Grenzwert für Scoring-Dimensionen Standardmäßig werden beim Scoring alle Korrelationen verwendet. Durch Auswahl des Felds und Eingabe der Anzahl der zu verwendenden kanonischen Korrelationen kann ein kleinerer Grenzwert festgelegt werden.

Anzeige Wählen Sie die Felder für die gewünschte Ausgabe aus. Bei den paarweisen Korrelationen handelt es sich um dieselben, die auch durch Ausführen der Prozedur für Korrelationen mit listenweisem Ausschluss fehlender Werte generiert würden. Dabei wird immer eine Tabelle mit Korrelationen und Tests erstellt.

Weitere Funktionen

Über dieses Dialogfeld wird Syntax für den Erweiterungsbefehl STATS CANCORR generiert. Führen Sie den folgenden Befehl aus, um die Syntaxhilfe für diesen Befehl anzuzeigen:

STATS CANCORR /HELP.

Anforderungen
